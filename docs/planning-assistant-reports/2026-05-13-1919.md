# 企劃助手報告 - 2026-05-13 企劃討論

結論：採用「一般戰鬥中的夜間替換」規則。黑影不是獨立於戰鬥之外的事件，而是夜晚或深夜森林戰鬥的特殊敵人：原本要發生森林戰鬥時，有 `33%` 機率把敵人替換成黑影。

這次聚焦範圍：
1. 森林夜晚的黑影遭遇
2. 夜晚 / 深夜戰鬥的 `33%` 黑影替換規則
3. 黑影與火把的唯一對應關係

我先採用的假設：
1. 黑影只在森林的夜晚與深夜出現，不進村、不進白天森林。
2. 黑影沿用一般遭遇戰流程與原本逃跑機制。
3. 黑影屬於山谷之神詛咒的外顯現象，不掉落一般戰利品。

建議規則草案：
- 觸發：
  夜晚或深夜在森林中原本會發生戰鬥時，有 `33%` 機率把原本敵人替換成黑影。白天不進行這個替換。
- 可用條件：
  黑影只會出現在森林場景，且限夜晚、深夜。村莊不出現。
- 消耗：
  使用火把時消耗火把 `1`。逃跑沿用原本戰鬥逃跑機制，不另開黑影專用規則。
- 結果：
  黑影的 `difficulty` 固定為 `99`，`fearedCombatTools` 只有 `torch`。一般武器、彈藥與其他戰鬥工具幾乎無法戰勝黑影；火把是唯一有效工具，選用火把可直接大勝利。
- 失敗或例外：
  黑影被擊敗後不掉落戰利品，`loot.common` 與 `loot.special` 都為空。若玩家沒有火把，仍照一般戰鬥結果與逃跑規則處理。
- 狀態歸屬：
  `33%` 夜間替換率、黑影難度、火把弱點與無戰利品規則屬於企劃與敵人資料；實際遭遇中的敵人 id、目前難度、遠程狀態等仍屬於存檔中的探索/遭遇狀態。

還需要你決定：
1. 黑影是否從第一天夜晚就會出現，還是要等主線揭露詛咒後才開啟。
2. 黑影的外觀與遭遇文案要不要隨森林層級或主線進度加深。
3. 火把擊敗黑影後，戰鬥結果文字要寫成「擊敗」、「驅散」還是「燒退」。

五條協助發想：
1. 黑影第一次出現時，可以讓描述強調「牠沒有清楚輪廓」，和普通森林敵人切開。
2. 火把擊敗黑影的文案可寫成火光逼散形體，避免像是普通物理擊殺。
3. 夜晚森林的補給提示可以讓賽拉或伊蓮暗示火把重要性，但不要直接說出 `33%`。
4. 黑影可作為第三本禁忌之書前的前兆，先讓玩家感受到「詛咒讓森林開始變得不對勁」。
5. 若之後要提高夜間壓力，可以讓深夜探索的普通遭遇率提高，但黑影替換率先固定維持 `33%`，避免規則太難讀。

建議寫入位置：已寫入 `docs/game-design.md` 的「森林探索與遭遇系統」、戰鬥工具規則與森林敵人戰利品區塊。之後若要同步 runtime，再新增黑影敵人資料與夜晚 / 深夜 `33%` 替換邏輯。
