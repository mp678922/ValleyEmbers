# 美術規格表

本文件記錄目前已確定的遊戲美術輸出規格。未明確指定的風格、動作、表情、配色與版本數量，後續再另外補充。

## 基本規格

| 類型 | 畫布尺寸 | 檔案格式 | 背景 | 單檔內容 | 命名規則 | 備註 |
|---|---|---|---|---|---|---|
| 角色人物 | `128 x 256 px` | `PNG` | 透明背景 | 1 個角色 | `character_<id>.png` | 角色尺寸固定，供人物立繪 / 角色圖使用 |
| 怪物 | `256 x 256 px` | `PNG` | 透明背景 | 1 隻怪物 | `monster_<id>.png` | 怪物尺寸固定，供遭遇 / 圖鑑 / 戰鬥顯示使用 |

## 共通交付規則

| 項目 | 規則 |
|---|---|
| 色彩模式 | `RGBA` |
| 背景處理 | 不可帶底色，需保留透明背景 |
| 檔案裁切 | 以規定畫布尺寸交付，不可輸出成其他尺寸 |
| 單位 | 規格中的 `128 x 256`、`256 x 256` 皆為像素 `px` |
| 檔案數量 | 每個角色 / 每隻怪物各自獨立成檔 |
| ID 原則 | 檔名中的 `<id>` 使用英文小寫與底線，例如 `character_mira.png`、`monster_forest_wolf.png` |

## 目前已定尺寸

- 角色人物：`128 x 256 px`
- 怪物：`256 x 256 px`
- 全部使用：`PNG`、透明背景
