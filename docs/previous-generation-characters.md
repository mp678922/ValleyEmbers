# 亡故上一代角色設計

本文件整理四位已確認亡故的上一代角色。用途是讓山谷村保留具體生活痕跡，並讓六名少女的責任、習慣與情感傷口有明確來源。

這四人不作為目前村莊常駐 NPC，也不進入普通村民交易、好感、排班或委託名單。她們/他們只存在於回憶、場景痕跡、舊物、墓碑、事件與角色對話中。調查隊失蹤者是否仍有倖存者，仍保留給主線後續處理；本文件只處理村中已確認亡故的人。

## 設計原則

- 每位亡故角色都要留下可被玩家看見的生活痕跡，而不是只出現在角色背景一句話裡。
- 每位亡故角色至少影響兩名現存少女，避免變成單一角色的私人物件。
- 死亡不只提供悲傷，也要解釋少女們現在為何會承擔某項工作、維持某個習慣，或避開某句話。
- 玩家第一次接觸這些痕跡時，不需要立刻知道完整死因。可以先看見物品、習慣或空位，再由角色互動逐步補上。

## 一、四位亡故角色

### 1. 瑪蓮，最後的舍監

- 身份：共同宿舍上一任舍監、伊蓮的老師，也是村中最後一位仍能支撐日常交接的上一代成員。
- 亡故時間：故事開始約一年前。
- 死因方向：病死。她長期照護宿舍、維持夜間點名與公共秩序，病情惡化後仍撐到最後一段交接完成。她的死亡不是神秘失蹤，而是讓六名少女真正意識到「再也沒有大人替她們收尾」的事件。
- 留下痕跡：宿舍點名簿、空床位旁的舊披肩、失物箱規則、飯碗數量、夜間門閂檢查順序、被伊蓮保存的舊故事索引。

與少女的關係：
- 伊蓮：直接繼承人。瑪蓮把宿舍、舊資料與「把殘酷事實講得讓人能聽下去」的方法交給她。伊蓮現在的舍監語氣、點名習慣與知識整理方式都來自瑪蓮。
- 艾妲：瑪蓮是艾妲母親失蹤後，少數能公開叫艾妲休息的人。艾妲現在會把會議記錄帶回宿舍整理，是因為瑪蓮生前要求「領袖的決定也要被別人看見」。
- 米菈：瑪蓮晚年病情讓米菈第一次真正面對「有些人不是治不好，而是太晚才被允許休息」。米菈對主角傷勢不肯放任，與這段經驗有關。
- 托莉：托莉小時候常被瑪蓮從開墾區邊緣抓回宿舍。她現在一聽見伊蓮用瑪蓮式的口氣喊她，就會心虛。
- 諾絲：瑪蓮留下很多宿舍修繕清單，諾絲嘴上嫌麻煩，卻一直照著修。
- 賽拉：瑪蓮曾把夜間異聲、點名缺漏交給賽拉判斷，形成現在伊蓮與賽拉的夜間情報合作習慣。

可演出痕跡：
- 主角第一次看見宿舍多出來的飯碗時，伊蓮可以淡淡說「以前收得太快會被她念」。
- 夜晚事件中，伊蓮照瑪蓮的舊順序檢查門閂，托莉忍不住小聲抱怨那套順序太嚴。
- 瑪蓮的空床位不可被主角使用；它是宿舍記憶錨點，不是可互動資源。
- 墓地對應 `嶄新的墓碑`。這是最後一位死亡的上一代成員之墓，銘文由伊蓮撰寫。

### 2. 葛倫，水渠與開墾區的老工頭

- 身份：開墾區與水渠維護者，曾帶村民修田埂、堆石牆、分配農務。
- 亡故時間：約三年前。
- 死因方向：老死。他是少數沒有被遺跡、黑影或瘴氣直接奪走的上一代成員，因此他的死對少女們來說既平靜又殘酷：不是所有離別都有敵人可恨，有些只是村莊真的老去了。
- 留下痕跡：開墾區舊水渠、磨損的木柄鋤、田埂邊的石標、過時但仍被托莉照著做的分工記號。

與少女的關係：
- 托莉：半個師父。托莉現在照顧田地的方式很多是學葛倫，但她只記得片段，因此常用「我知道啦」掩飾自己其實怕忘記。
- 賽拉：葛倫生前常提醒她巡邏不能只看森林，也要看水渠、田埂與村口退路。她對托莉嚴厲，不是看不起托莉，而是知道開墾區邊緣一旦出事，最先被推到前面的人往往是最想證明自己的孩子。
- 艾妲：葛倫生前常用很直接的話提醒艾妲「村長不能只看最急的火，也要看明年還有沒有水」。艾妲現在安排巡村時會特別看開墾區與水源。
- 諾絲：葛倫留下的工具粗糙但耐用，諾絲尊重他的實用派做法。她修工具時常罵「這種綁法也太老派」，但不會改掉關鍵結構。
- 米菈：曾照顧葛倫臨終前的身體狀況。這讓米菈第一次明白，有些照護不是把人治好，而是讓人最後一段路不要那麼痛。

可演出痕跡：
- 開墾區可有一段主角視角：田埂上某些石標位置過於規整，像有人直到老得走不穩，仍想讓後人少走錯一步。
- 托莉整理水渠時會一邊抱怨舊標記難懂，一邊照得很仔細。
- 賽拉看見托莉跨過葛倫留下的石標時，語氣會立刻變硬。
- 墓地對應 `風化的墓碑`。銘文由托莉撰寫，保留葛倫和開墾區、水渠記號的關係。

### 3. 艾芙拉，藥草棚的舊醫者

- 身份：上一代藥草師，米菈的祖母以外，實際在村中長期照護病患的人。她把藥棚日常交給米菈，讓米菈不只是懂藥草，也懂怎麼照看人。
- 亡故時間：約兩年前。
- 死因方向：採藥受傷後失聯。她外出採藥時受傷，最後沒能回到村裡；村民只找到部分採藥工具與血跡，因此村中將她視為亡故。她的失聯讓米菈提前成為村中唯一能穩定處理傷病的人，也讓米菈對獨自外出採藥格外謹慎。
- 留下痕跡：藥瓶擺放順序、舊藥方、按病況記錄的布籤、米菈仍不敢丟掉的空藥罐。

與少女的關係：
- 米菈：醫療責任的來源。艾芙拉教她藥草、包紮與看人臉色，也教她不要把「我沒事」當真。艾芙拉採藥失聯後，米菈不只接下藥棚，也背著「如果當時準備再多一點，也許她能回來」的想法。
- 伊蓮：艾芙拉會要求伊蓮記錄宿舍夜間咳嗽、發燒與夢囈，讓伊蓮的記錄習慣不只服務故事，也服務照護。
- 艾妲：艾芙拉生前常反對艾妲把藥草優先讓給別人後自己硬撐。艾妲現在被米菈念時，會想起艾芙拉。
- 賽拉：賽拉受傷後最討厭進藥草棚，因為艾芙拉曾經看穿她害怕失去姊姊。艾芙拉失聯後，賽拉對採藥路線也多一層戒心；米菈繼承艾芙拉的敏銳，讓賽拉更難隱瞞傷勢。
- 諾絲：藥棚的晾架與製藥台有一部分是諾絲照艾芙拉留下的尺寸修的。諾絲不愛承認，但她修醫療器具時會比平常更安靜。

可演出痕跡：
- 米菈整理藥瓶時，某些瓶子明明空了仍留在原位，因為那是艾芙拉教她辨認病況時用的樣本。
- 伊蓮的夜間知識話題可以偶爾提到「以前藥棚那邊會要我記誰睡得不安穩」。
- 艾芙拉沒有完整墓葬；藥草棚旁可以有一塊簡單名牌與固定更換的乾草，表示村民已把她視為亡故，卻仍保留她可能只是沒能回來的沉默。
- 墓地對應 `生苔的墓碑`。這不是完整墓葬，而是村裡替她留下的紀念碑；銘文由米菈撰寫。墓碑邊每天可取得 `乾苔 x1`，但玩家可見文字不應把它寫成普通採集點。

### 4. 羅恩，巡路人與舊信使

- 身份：艾妲的父親。村口、森林邊緣與外界路線的巡路人，也是上一代中最熟悉外部道路的人。
- 亡故時間：約四年前。
- 死因方向：為了救賽拉被黑影殺死。賽拉曾在夜間或接近夜晚時陷入黑影威脅，羅恩把她推出危險範圍，自己被黑影吞沒。村人找回的只剩染血斗篷、破裂路標與他留下的巡路痕跡。這件事是賽拉對夜晚森林、退路與裝備規矩極端嚴格的核心原因。
- 留下痕跡：村口舊路標、巡路繩結、外界地名殘頁、賽拉沿用的安全路線記號。士兵長槍其實也是羅恩的遺物，但這條線索應藏得很深，早期不要在道具來源、賽拉教學或一般場景描述中明說。

與少女的關係：
- 賽拉：巡邏技術來源之一。她不是羅恩的親人，卻從他那裡學到如何看折枝、泥痕與路標角度。羅恩為了救她被黑影殺死，讓她相信「不確認退路就外出」不是愚蠢，而是會把別人拖進死地。她也背著一層不容易說出口的愧疚：救她的人是艾妲的父親。她對士兵長槍會有不明說的慎重態度，但早期只表現為要求主角正確使用裝備。
- 諾絲：羅恩常把損壞的路標和扣具丟給諾絲父親修。諾絲現在維修外出裝備時，會沿用羅恩留下的結繩方式；若修士兵長槍，她會比修一般武器更安靜，但不需要直接說出原因。
- 艾妲：羅恩是她的父親，也是她理解「村莊不能只看內部」的來源。母親是前任村長與調查隊領導者，父親是巡路人；兩人的缺席讓艾妲同時承接村內領袖責任與村外斷聯壓力。羅恩為救賽拉而死這件事，不應讓艾妲怪罪賽拉，但會讓她更難在安全問題上只說一句「不是你的錯」。
- 托莉：托莉小時候聽過羅恩講外面的路，因此她對探險和外界有想像。她夢想成為探險者，不只是叛逆，也是在追逐一個已經沒人能證明的外部世界。
- 伊蓮：伊蓮保存羅恩帶回的外界地名殘頁。這讓她知道山谷不是世界全部，也讓她更容易理解主角作為外來者的意義。

可演出痕跡：
- 村口路標可以有一道舊裂痕，賽拉每次經過都會確認它有沒有被動過。
- 士兵長槍可以放在舊武器庫或賽拉管理的外出裝備中；主角取得時，只讓賽拉確認主角是否知道長槍不是拿來逞強的東西，不提羅恩。羅恩關聯與「他是艾妲父親」這層關係，適合留到後期信任事件、舊武器修復事件或巡路真相揭露後才回收。
- 托莉問主角外面的世界時，可能提到羅恩以前說過的某個地名，但她已經不確定那是不是故事。
- 羅恩死亡真相不應太早揭露。早期只讓玩家知道他死於一次夜間危險，後期再揭露他是為了救賽拉才被黑影殺死，讓賽拉的嚴厲從「過度警戒」反轉成「不想再讓任何人替自己死一次」。
- 墓地對應 `老舊的墓碑`。銘文由艾妲撰寫，早期只呈現巡路人與回家方向，不揭露父女關係、救賽拉真相或士兵長槍關聯。

## 二、與六名少女的關係總表

| 少女 | 主要亡故連結 | 形成的現在性格或責任 |
|---|---|---|
| 艾妲 | 瑪蓮、葛倫、艾芙拉、羅恩 | 她的領袖責任不是只來自母親失蹤，也來自父親羅恩死亡後留下的村外斷聯壓力，以及這些上一代逐一離開後留下的公共工作。她會把每個空位都轉成待辦事項。 |
| 米菈 | 艾芙拉、瑪蓮、葛倫 | 她的醫療責任來自親眼看見大人太晚被照顧；她不接受「我沒事」這種說法，是因為她知道那常常是最後的逞強。 |
| 諾絲 | 葛倫、羅恩、瑪蓮、艾芙拉 | 她修的不只是物件，而是上一代留下來仍勉強能用的生活秩序。她嘴硬，是因為承認某件工具有感情會讓工作變慢。 |
| 賽拉 | 羅恩、葛倫、艾芙拉、瑪蓮 | 她的警戒來自具體死亡經驗，不是單純多疑。羅恩為了救她而死，且羅恩是艾妲的父親，讓她面對艾妲時總有一層壓住不說的愧疚。她知道邊界、退路、夜間點名都曾經救過人，也曾經太晚救到人。 |
| 托莉 | 葛倫、羅恩、瑪蓮 | 她想成為探險者，不只是想逞強，而是想追上那些曾經把村子連到田地、道路與外界的人。她怕自己只被留下來做小孩能做的事。 |
| 伊蓮 | 瑪蓮、艾芙拉、羅恩 | 她保存的不只是傳說，也保存村莊如何活下去的細節。她用故事說話，是因為直接念名字會讓每個人都停下來。 |

## 三、場景使用規則

- 宿舍、藥草棚、工坊倉庫、開墾區、村口與墓地都應各自承載至少一位亡故角色的痕跡。
- 亡故角色的名字不必在每次場景描述中出現；更多時候用物品、動作與少女反應暗示。
- 需要避免把四位亡故角色都寫成完美大人。她們/他們可以固執、偏心、失誤、太會忍耐，這樣少女繼承的習慣才會有複雜度。
- 玩家可逐步從少女對同一位亡故角色的不同記憶中看見矛盾：有人懷念，有人埋怨，有人不敢講，有人只留下動作。

## 四、五條協助發想

1. 讓主角第一次在宿舍看到多出來的飯碗時，伊蓮只說「那個位置先別收」，不立刻解釋瑪蓮。
2. 設計托莉修水渠的小事件，她照葛倫的舊石標做事，卻不知道某個標記其實是「不要再往外」。
3. 讓米菈治療主角時，下意識重複艾芙拉的老話，說完才發現自己越來越像那位舊醫者。
4. 讓賽拉教主角看路標時提到羅恩，但語氣像在講規則，不像在講人；等好感提高後才承認那是她學會巡路的來源。
5. 終局前村莊會議可以讓六名少女各自把一件亡故角色留下的物品放到桌上，表示她們不是替上一代受苦，而是帶著留下來的東西重新選擇山谷。
